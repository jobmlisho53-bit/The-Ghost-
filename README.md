# Ghost YouTube

An AI-native video platform that generates on-demand content based on user requests and trending topics. All content is AI-generated with no human creators involved, providing unlimited content diversity.

## Features
- AI-generated videos on any topic (except immoral content)
- Real-time content generation
- Trend-based content suggestions
- Personalized user experience
- No traditional social media elements (likes, views, comments)

## Tech Stack
- Node.js/Express for backend
- React for frontend
- TensorFlow/PyTorch for AI models
- PostgreSQL for database
- Redis for caching
- Docker for containerization

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables
4. Run the application: `npm run dev`
